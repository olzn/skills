# Suites

Composed skill systems that contain multiple related skills, references, and installation scripts.

## Available Suites

- [`ui-craft`](ui-craft/) - A coordinator skill plus surface and system domain skills for web interface work.
- [`hig`](hig/) - Three workflow skills (`hig` lookup and routing, `hig-design` build-time guidance, `hig-review` audits) that keep iOS and macOS designs and prototypes aligned with Apple's Human Interface Guidelines. Liquid Glass-current snapshot with provenance-marked metrics, stale-prior corrections, and scripts that make staleness detectable. The index resolves every iOS/macOS-relevant HIG slug — distilled corpus sections, API-symbol routes for code-phrased questions, and live-fetch routing for niche topics.

## Boundaries

The suites co-own HTML prototypes that imitate native Apple UI: ui-craft owns the web mechanics, hig owns the native idiom and metric values — on conflict, HIG wins for native-targeting work.
